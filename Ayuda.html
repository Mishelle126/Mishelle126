Etiquetas del HTML

  • H1… H6 Para títulos dentro del contenido.

  • Head Parainformación sobre el documento.

  • Body Para el contenido.

  • div División dentro del contenido.

  • a Para enlaces.

  • b o el strong Para poner el texto en negrita.

  • br Para saltos de línea.

  • img Para añadir imágenes al documento.

  • span Para estilos de una parte del texto.

  • iframe Define una ventana o frame interno.

  • input Añade un cuadro de entrada de texto a un formulario.

  • label Define una etiqueta relacionada a un elemento imput de un formulario.
